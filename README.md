# TothSidleck2026
Public Data and Code repository for Toth, Sidleck, & Eldo 2026 from the Insanally Lab.

## Requirements

All packages and scripts used to analyze our data were custom written in `python` (3.13.13) using `jupyterlab` (4.5.6) and `ipython` (9.11.0).

Additional packages required are:

- `matplotlib` (3.10.9)
- `numpy` (2.4.4)
- `pandas` (3.0.2)
- `scipy` (0.19.1)
- `pingouin` (0.6.1)
- `scikit-learn` (1.8.0)
- `scipy` (1.17.1)
- `tqdm` (4.67.3)

## Files and Directories

- **`Readme.md`:** This file
- **`InsanallyLabEphysTools/`**: Directory containing custom written scripts to help with statistics, analysis, and plotting. 
- **`Notebooks/`:** Directory containing Jupyter notebooks used to regenerate individual figure panels from the manuscript. Notebook filenames correspond to the associated figure panel labels (e.g., Figure2b.ipynb corresponds to Figure 2b). To regenerate a figure panel, run each cell in the notebook in order from top to bottom.
- **`Source Data Files/`**: Directory containing source data needed to regenerate figures.
